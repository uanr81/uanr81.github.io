Общее введение в пользовательский интерфейс

FontForge имеет одинаковый интерфейс в операционных системах Windows, Mac OS и GNU/Linux.

Вот краткое введение в основные функции, сделанное Дэйвом Кросслендом на семинаре Crafting Type, некоммерческой мастерской по шрифтовому дизайну, которая поддерживает проект FontForge:

Есть 4 основных окна:

1. Окно просмотра шрифтов

Это окно появляется при первом запуске FontForge. Показывается таблица глифов в шрифте.

2. Окно Font View

Дважды щелкнуть глиф в Font View, чтобы открыть его в Character View. Здесь можно рисовать и редактировать глифы либо по одному с помощью интерфейса вкладок, либо рядом с подготовленным текстовым файлом слов для пошагового прохождения.

3. Окно Metrics View

Переход к Window, Metrics, или Metrics, Window. Выбор несколько глифов в Font View и нажать Ctrl + K. Также можно перетаскивать глифы из Font View в Metrics View.

Здесь можно настроить интервал и кернинг шрифта. Здесь можно просмотреть подготовленный текстовый файл со списком слов.

4. Окно Font Info Window

Перейти по Elements, Font Info. Здесь найти всю информацию metadata о шрифте.

5. Среда набора текста для тестирования

Помимо любого редактора шрифтов, понадобится система набора текста для проверки используемых шрифтов. Это один из big secrets профессиональных шрифтовых дизайнеров: большая часть наиболее важной работы в проекте дизайна шрифта выполняется вне редактора шрифтов, при тестировании документов, созданных в системе набора текста. Основными системами являются:

  • Web (Firefox, Chromium, и подобные)
  • DTP инструменты (Scribus, Inkscape, и подобные)
  • Текстовые процессоры (Libre Office, AbiWord, Calligra Words, и подобные)
  • Программирование процессоров макета страницы (LaTeX, ConTeXt, SILE, ShoeBot, Even, и подобные)
