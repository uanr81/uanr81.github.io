Установка FontForge

FontForge — бесплатное программное обеспечение, поэтому можно загружать, делиться и устанавливать копии без каких-либо ограничений на использование — приветствуется как коммерческое, так и личное использование. Это приложение поддерживается сообществом, и каждый может внести свой вклад в исходный код.

FontForge доступно в виде простых в установке пакетов для операционных систем Windows, Mac OS X и GNU/Linux.

Установка FontForge в Windows

Инструкция по установке доступно для официальных сборок Windows, подготовленных Джереми Таном.

Установка в GNU/Linux

Самый простой способ установить FontForge на ваш компьютер с Linux — использовать репозиторий пакетов дистрибутива Linux.

Debian или Ubuntu

Пакет FontForge, включенный в Ubuntu 14.04, по умолчанию датируется 2012 годом, поэтому предпочтительнее установить более свежий пакет из FontForge. [Архив личных пакетов (Personal Package Archive)(PPA)] (https://launchpad.net/~fontforge/+archive/ubuntu/fontforge).

Убедиться, что установлен вспомогательный скрипт add-apt-repository:

sudo apt-get install software-properties-common;

Добавить FontForge PPA (что также добавит ключ аутентификации):

sudo add-apt-repository ppa:fontforge/fontforge;

Обновить список программного обеспечения, включив в него пакеты из PPA:

sudo apt-get update;

Установка FontForge:

sudo apt-get install fontforge;

Fedora

Чтобы установить FontForge на настольный компьютер с Fedora Linux, выполнить следующую команду dnf от имени пользователя root. Для завершения потребуется около 10 МБ загрузки.

sudo dnf install fontforge

Если не скомпилировать программное обеспечение на компьютере с Fedora, после установки gcc, automake, autoconf и других можно получить ошибку во время выполнения autogen.sh с помощью libtoolize. В этом случае может потребоваться установить пакет libtool-ltdl-devel в Fedora или аналогичный пакет разработки в другом дистрибутиве GNU/Linux.

После установки yum запустить FontForge из меню или непосредственно из консоли или терминала gnome, вводом команды fontforge.

Скомпилировать собственную версию с GitHub.

GitHub — служба хостинга исходного кода, где каждый может внести вклад в разработку программного обеспечения. Хранит текущее состояние разработки приложения. В некоторых случаях, возможно, потому, что вам нужен доступ к функции, еще не доступной в пакетах выпуска, можно пожелать скомпилировать собственную версию с GitHub.

Полная инструкция находится на https://github.com/fontforge/fontforge/blob/master/INSTALL.md
